CT607 – Introduction to Data Science

Course Objective(s):

  1. To explore Data science using Python programming

Introduction to Data Science – Evolution of Data Science – Data Science Roles – Stages in a Data Science Project – Applications of Data Science in various fields – Data Security Issues.

Data, Open, Social Media and Multimodal data - Structured and unstructured data, -Data Collection Strategies – Data Pre-Processing – Data Cleaning – Data Integration and Transformation – Data Reduction – Data Discretization - Dataset generation.

Exploratory Data Analysis, Data reshaping, Filtering, Cleaning, Formatting – Descriptive Analysis – Diagnostic Analysis – Predictive, Prescriptive and Mechanistic analysis.

Basic Visualization Tools, Specialized Visualization Tools, Interactive Data Visualization, Matplotlib, Seaborn, Plotly.

Data Wrangling, Web scrapping, Combing and merging data sets, Reshaping and pivoting, Data transformation, Learning models using Sci-Kit Learn, Ethics of big data, Data Journalism

Course Outcomes :

Students will be able to

  1. Understand the basic process of Data Science using Python.
  2. Use python packages for Data Wrangling
  3. Use python packages for Data visualization
  4. Use python packages for Learning and Intelligence
