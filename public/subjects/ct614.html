CT614 - Deep Learning and Techniques

Course Objective(s):

  1. To understand the principles of Deep Learning
  2. To experiment the various frameworks available for Deep Learning
  3. To develop different deep neural network models
  4. To demonstrate the use of Transfer Learning techniques for real-time usecases

Introduction to Deep learning – Data Loading - Learning models – Backpropagation – Optimization of Deep learning models – Normalization Techniques - Hyperparameters – Selection and Tuning – Regularization – Unsupervised representation learning – Self-Supervised Learning

Deep Learning Frameworks - Pytorch, Tensorflow - Build your first Model - Data preparation – Augmentation – Train/Dev/Test Distributions - Dev/Test Sets and Metrics - Error Analysis – Avoidable Bias – Model Stacking – Multi-task Learning – Transfer Learning

Foundation Layers of CNN – Stacking of Neural layers - Image CNNs – Pre-trained Models (GoogleNet, AlexNet, VGG, Yolo, ResNet) - Introduction to Recurrent Neural Networks (RNN) – LSTM – GRU – Bidirectional RNNs – NLP – Word Embedding – Sentiment Analysis – Named Entity Recognition – Neural Machine Translation – Attention Mechanism – Pre-trained Models (Google’s BERT, Open AI’s GPT3, Facebook’s RoBERTa)

Generative Adversarial Networks – Loss function – Conditional GANs - Cycle GAN – Applications of GAN - Graph Neural Networks - Graph Convolutions – Graph Auto Encoders- Spatio Temporal Graph Neural Networks - Deep Graph Library

Deep Learning at Scale – Building Model Service – ONNX - NVIDIA NGC - Dockers – Inference Serving – Concurrency - Performance Optimization – Automatic Mixed Precision - Container Orchestration - Model Deployment – Types of Deployment - Edge Deployment

Course Outcomes :

Students will be able to

  1. Discuss the principles of Deep Learning and its applications
  2. Use Deep Learning Frameworks to build Deep Learning Models
  3. Experiment various pre-trained models for Image and Language processing
  4. Explain the Advance Deep Learning Models
  5. Develop an Deep Learning pipeline to ship DL model as Product
