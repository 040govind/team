CT621 - Natural Language Processing

Prerequisite: Machine learning Techniques /Deep Learning Techniques

Course Objective(s):

  1. To introduce language processing techniques to enable Text data processing
  2. To impart knowledge on text data processing using Statistical and Machine learning models
  3. To introduce the various Embeddings and Deep learning models for NLP

Natural Language Processing– Applications of NLP- Linguistic Background - NLP tasks - Ambiguities in NLP tasks -Regular Expressions - Linguistics resources- Corpus, WordNet, - Text Normalization - Edit Distance - Tokenization -Stemming-Lemmatization.

Morphological Analysis - Part of speech tagging - Shallow parsing – Dependency parsing - Semantics - Synonymy – Polysemy - Similarity measures – Ontology - Coreference Resolution - Text coherence.; Tools–Natural Language Toolkit, Stanford CoreNLP.

Language model- n-gram Language models- Hidden Markov Model– Conditional random Fields- Topic models - Graph Models

Machine Learning for NLP: Text representations- Vector space model- Language Features-Maximum Entropy classifier – Document Clustering

Deep Learning for NLP: Embeddings- Recurrent Neural networks - Encoder Decoder - Attention - Transformer based models – BERT - XLNET - GPT - T5- Multilingual models – mBERT- XLM ; Tool: PyTorch. Applications and Case Studies- Machine Translation - Sentiment Analysis -Conversational Agents- Intent Classification

Course Outcomes :

Students will be able to

  1. Identify the patterns in text and pre-process the large text corpus
  2. Describe and work with basic NLP tasks
  3. Use statistical and machine learning models to solve NLP problems
  4. Adopt Deep learning models to solve NLP problems
