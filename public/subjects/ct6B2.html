CT6B2 - Computer Vision

Course Objective(s):

  1. To understand the fundamental concepts related to image processing, feature extraction, motion analysis etc.
  2. To apply the concepts to solve computer vision problems of different fields.

Introduction to Computer Vision and Basic Concepts of Image Formation- Image Transforms-Image Enhancement-Image Filtering.

Image Descriptors and Features: Texture Descriptors-Colour Features-Edges/Boundaries. Object Boundary and Shape Representations-Interest or Corner Point Detectors-Histogram of Oriented Gradients-Scale Invariant Feature Transform-Speeded up Robust Features .

Image Segmentation: Region Growing, Edge Based approaches to segmentation, Graph-Cut, Mean-Shift, Texture Segmentation.

Motion analysis: Background Subtraction and Modeling- Optical Flow- KLT- Spatio-Temporal Analysis - Object tracking: Mean shift tacking

Image classification-Object detection-content based image retrieval-Industrial Applications.

Course Outcomes :

Students will be able to

  1. Implement fundamental image processing techniques required for computer vision
  2. Implement image segmentation and object tracking techniques.
  3. Apply features and descriptors for image classification and object detection.
  4. Develop applications using computer vision techniques.
