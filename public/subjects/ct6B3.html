CT6B3 - CLOUD AND EDGE COMPUTING

Course Objective(s):

  1. To understand Cloud Computing concepts, technologies, architecture and applications
  2. To understand different cloud programming platforms and tools to develop and deploy applications on cloud
  3. To understand edge computing architecture and edge analytics

Overview of Computing Paradigms- Distributed Systems Models and Enabling Technologies - Cloud Computing Properties and Characteristics - Business Drivers for Adopting Cloud Computing

Introduction to virtualization - Different Approaches to Virtualization - Server, Storage, Network Virtualization - Virtual Machine Provisioning and Manageability - VM Placement-VM Migration - Hypervisors - Case studies: VMware, KVM, Xen

Service Science - Service Oriented Architecture - Web Services: SOAP, WSDL, UDDI - Web Services Discovery and Composition - REST based Web Services

Cloud Computing Architecture - Cloud Computing Service Delivery Models: Infrastructure as a Service (IaaS), Platform as a Service (PaaS), Software as a Service(SaaS) Deployment Models: Public Cloud, Private Cloud, Hybrid Cloud - Data Center Design and Management - Service Level Agreements (SLAs) –Cloud Security Risks - Case Studies: Amazon AWS, Microsoft Azure, Amazon EC2, Google Cloud – Analytics in/for Cloud

Edge computing Concepts - Reference Architecture - Edge Computing in the IoT – Edge Analytics and Data Management - Use Cases

Course Outcomes :

Students will be able to

  1. Acquire Knowledge on the concepts and technologies of Cloud Computing.
  2. Define the principles of virtualization
  3. Identify the Service Oriented Architecture for Distributed Computing workflow.
  4. Acquire Knowledge on the concepts and technologies of Edge Computing.
