CT6C1 - High Dimensional Data Analytics

Course Objective(s):

  1. To understand the attributes of High Dimensional Data
  2. To introduce about the High Dimensional Healthcare data & Geospatial data
  3. To explore the principles of accelerated GPU Data Science
  4. To experiment case studies using popular open source frameworks

High Dimensional Statistics – Basic, Matrix Concentration Inequalities – Gradient Descent - Rates of Convergence and Acceleration - Constrained Optimization - Block Coordinate Descent and ADMM – Blessings of High Dimensionality - High-Dimensional Geometry – Scalable Optimization: Convex and Nonconvex – Convex Methods for Sparse Signal Recovery and Low-Rank Matrix Recovery - Decomposing Low-Rank and Sparse Matrices

Types of Health care Data - Data Preparation – Data Classification – Time Series Classification - Multi-level feature extraction - High-dimensional pattern regression – Federated Learning – Deep Learning in EHR - Permutation Feature Importance (PFI), Local Interpretable Model-agnostic Explanations (LIME) - SHapley Additive exPlanation (SHAP) - Class-Activation Mapping (CAM) and Gradient-Weighted CAM – Clinical Decision Support System

Understanding Spatial Data – Spatial Analysis – Geographic Information Science and Systems – Geospatial Visualization - Matplotlib, Basemap, Cartopy, Mplleaflet and Folium libraries - Google Maps API – GeoPandas – Geospatial Object Manipulation - Shapely and RasterIO libraries – Hadoop paradigm for Taming big geospatial data – CyberGIS

GPU Parallelism – Data Processing Evolution – RAPIDS – End-End RAPIDS workflows - Data Cleaning – Feature Engineering - Dataframes: cuDF – Scaling: dask-cudf – cuML – Satellite Data Processing – cuSpatial - SatPy – Datascience workflow on CPU, GPU, Multi-GPU – Accelerating Apache Spark with Rapids

Explore features of Cloud Healthcare API and use BigQuery for analytics, Google Cloud - MIMIC-III Dataset: Preprocessing and imputation of MIMIC III data, EHR Encodings for machine learning models - Visualize BigQuery Geospatial Data using Google Maps API, Visualize Real Time Geospatial Data using Google Data Studio

Course Outcomes :

Students will be able to

  1. Discuss the principles behind High Dimensional Data
  2. Use Deep learning libraries to process Electronic Healthcare Records
  3. Perform Geo-spatial Data Analysis
  4. Apply the Accelerated GPU Data Science paradigm
  5. Explore the case studies using Google Cloud APIs
